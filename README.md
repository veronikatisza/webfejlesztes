# webfejlesztes
A webfejlesztés tárgy beadandóa.
Készítő neve:
Tisza Veronika

A működő weboldal URL címe:
https://veronikatisza.github.io/webfejlesztes/

A weboldal témájának rövid leírása:
A weboldalam a Victoria’s Secret történetét és modelljeit mutatja be, valamint egy kvízzel és egy nyereményjátékkal teszi interaktívabbá az élményt.

A saját JavaScript kódok helye és működésük leírása:

    Az elkészített weboldalamhoz 2 JavaScript kód is tartozik:

        1. script.js - néven (TiszaVeronika/Js/script.js könyvtárban)
        Feladata a nyereményjáték űrlap validálása, hibák kijelzése, valamint az űrlap beküldésének megakadályozása hibás kitöltés esetén.

        2. scriptquiz.js - néven (TiszaVeronika/Js/scriptquiz.js könyvtárban)
        Feladata a quiz kiértékelése, eredmények kiszámítása.

A használt egyedi fonttípus és annak helye:

    Az általam készített weboldalon két Google Fonts betűtípus került felhasználásra: 

        1. "Dancing Script" 
        2. "Playwrite NO" 
    Mindkettő a <link> tageken keresztül, a head szekcióban kerül betöltésre.
